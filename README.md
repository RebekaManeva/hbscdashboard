# HBSC Македонија — Дигитален Dashboard за Добросостојба на Млади

Интерактивна веб-апликација базирана на податоците од меѓународното истражување HBSC (Health Behaviour in School-aged Children), која ја визуелизира добросостојбата на млади на возраст 11–15 години во Македонија.

Апликацијата овозможува преглед на клучни теми — стрес, булинг, спиење и употреба на екран — преку интерактивни графици и споредби со регионални земји. Исто така, вклучува личен прашалник каде корисникот може да ги внесе своите навики и да добие персонализирана анализа во споредба со врсниците.
